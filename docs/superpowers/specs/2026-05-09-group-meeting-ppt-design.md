## 各页详细内容

### 封面
- 题目："纯镍退火工艺的贝叶斯优化——数据预处理与建模"
- 副标题：IPF/GND 特征量化 + GPR 代理模型

### 背景（2 页）
- 页2：纯镍退火问题——温度 800-1400°C、H₂/Ar 气氛、时间控制；目标：定向织构产率最大化
- 页3：传统试错法痛点 vs 数据驱动 BO 的优势（样本效率、不确定性量化、物理约束可集成）

### 模块一：数据预处理（4 页）
- 页4：原始数据——AZtecCrystal EBSD 导出格式（IPF Coloring + GND Density 逐像素）；数据规模 10⁴-10⁵ 像素/样本
- 页5：IPF 编码原理——对称归约 + 立体投影 + 重心坐标 → 6 维 RGB 统计特征（均值/标准差）
- 页6：GND 降维——15 维统计特征（集中趋势、离散度、分位数、分布形状、极端区域比例）；物理动机（再结晶驱动力/形核点/已有低缺陷区域）
- 页7：压缩总结——4N 标量 → 21 维特征向量；丢失 vs 保留信息对比；Pearson 初筛

### 模块二：贝叶斯优化（7 页）
- 页8：BO 循环总览图——代理模型（GPR）→ 采集函数（EI）→ 建议新实验 → 更新模型
- 页9：GPR 代理模型（上）——特征空间（34 维）→ 核函数 `ConstantKernel * Matern(nu=2.5, ARD=True) + WhiteKernel` → 为什么 Matern 2.5 适合物理过程（二阶可微的平滑性 + 更真实的粗糙度）
- 页10：GPR 代理模型（下）——ARD 自动相关性确定机制：每个特征独立的长度尺度 l_k → 小 l_k = 高重要性 → 自动识别关键预测因子；WhiteKernel 处理观测噪声；长度尺度边界 0.3-5.0 的设计理由
- 页11：EI 采集函数（上）——Exploration vs Exploitation 的核心权衡 → EI(x) = E[max(f(x) - y_best, 0)] → 利用预测均值高区域 + 探索高不确定区域
- 页12：EI 采集函数（下）——局部 y_best（同织构方案内 Winsorized P95 避免单点异常）→ 两步优化：10 万 LHS 随机采样粗搜 + Top-5 L-BFGS-B 梯度精炼
- 页13：多任务与流程——4 种织构目标方案（multi-hot encoding）→ 完整优化流程图
- 页14：实验结果——LOOCV parity plot + ARD 特征重要性 + 不确定性收敛曲线
