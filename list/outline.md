# 纯镍退火工艺的贝叶斯优化——数据预处理与建模

## 研究背景
### 退火工艺参数共同决定织构产率，传统试错法面临四维搜索瓶颈
- 退火参数：温度 800–1400°C × 保温时间 1–30h × H₂/Ar 气氛比例——共同构成连续四维搜索空间
- 试错法痛点：单次实验周期 ~12h，无法系统遍历空间；每次仅能凭经验调整1-2个参数，难以找到真正全局最优
- 数据稀缺：当前仅约11组有效实验数据点，N远小于特征维度，传统响应面回归建模失效
- 目标：以最少实验次数，找到使定向织构产率最大化的工艺参数组合

### 贝叶斯优化以样本效率、不确定性量化和物理约束集成突破传统瓶颈
- 样本效率：25–50个实验即可收敛，相比传统响应面法减少60%以上实验次数——原因在于每一步都学习全局函数结构
- 不确定性驱动搜索：高斯过程回归提供每个预测点的方差，采集函数自动平衡"探索高不确定区域"与"利用已知高产率区域"
- 自动特征选择：ARD Matern核为每个特征学习独立长度尺度，无需人工指定"哪些特征对产率影响大"
- 工作流程：EBSD数据量化降维 → 高斯过程代理模型构建 → EI采集函数驱动迭代优化 → 实验验证

## 数据预处理：IPF / GND 量化降维
### 原始数据：AZtecCrystal EBSD扫描包含逐像素取向色彩与位错密度
- 每样本对应一个实验文件夹，含 pre.csv（退火前EBSD扫描）、done.csv（退火后）、condition.xls（工艺参数）
- AZtecCrystal CSV格式：2行元数据表头 + N行逐像素数据（Index, X, Y, IPF Coloring, GND Density）
- IPF Coloring列：空格分隔的R G B三整数（0–255），由晶粒取向经立体投影+重心坐标计算——本质是取向的二维色彩编码
- GND Density列：几何必要位错密度（10¹⁴/m²），退火驱动力来源；存在少量缺失值需清洗
- 数据规模：单次扫描10⁴–10⁵像素，需压缩至样本级特征向量才能作为模型输入

### IPF色彩编码的数学原理：从晶粒取向到RGB的四个步骤
- [image: ipf_encoding.png]
- Step 1 — 对称归约：立方晶系对称性使得任意取向向量 (h,k,l) 可归约至 [001]–[101]–[111] 标准极图三角形内
- Step 2 — 立体投影：X = x/(1+z), Y = y/(1+z)，将三维球面方向压缩至二维平面
- Step 3 — 重心坐标插值：在P_001(红)、P_101(绿)、P_111(蓝)三个顶点间求解权重——RGB由此而来
- Step 4 — 亮度拉伸：max(R,G,B)拉满至255，确保每个取向对应最饱和颜色（AZtec标准处理）
- 降维策略：逐像素RGB → 6维统计特征（每通道均值μ+标准差σ）
- 物理含义：均值刻画整体织构取向偏好（如高R_Mean→[001]织构优势），标准差反映取向分散程度（低→集中织构，高→随机取向）

### GND密度分布的15维统计量化：四层特征覆盖退火动力学信息需求
- 位错密度是再结晶驱动力来源——分布的形状、尾部极端值比均值本身更具预测价值
- 集中趋势（2维）：Mean（整体变形程度）、Q50（中位数，对异常值稳健）
- 离散程度（3维）：Std（绝对离散）、IQR（Q75−Q25，稳健离散）、CV（σ/μ，归一化不均匀度，消除量纲效应）
- 分位数尾部（5维）：Q25→Q75→Q90→Q95→Q99——系统覆盖低缺陷到极端高位错的全范围，Q95/Q99代表潜在优先形核点
- 分布形状（3维）：Peak（0.999分位数锚点）、Skewness（偏度：正值=多数低GND+少数极高GND=典型冷变形态）、Kurtosis（峰度：高值=尾部更厚=极端像素更集中）
- 极端区域比例（2维）：HighRatio（GND>Q75+1.5×IQR的面积占比=高变形储能区）、LowRatio（GND<Q25占比=已回复区域）
- 设计逻辑：这四层特征系统回答——再结晶驱动力多强？形核点在哪里？已有多少低缺陷区域？微观结构有多均匀？

### 压缩路径总结：10⁴像素→21维特征，保留分布统计轮廓
- [image: feature_compression.png]
- 压缩比约10⁴:1：N像素×4通道 → 6维IPF统计 + 15维GND统计 → 21维Pre_特征向量
- 丢失信息：像素空间位置（X,Y坐标）、空间关联（相邻像素关系）、单个晶粒边界、EBSD花样质量等辅助指标
- 保留信息：织构宏观取向偏好（均值）、取向分布弥散度（标准差）、GND分布的完整统计轮廓（位置+尺度+形状+尾部）
- Pearson初筛：训练样本<特征数×5时，对|r|>0.9的特征对删除冗余项——进一步降维，缓解小样本过拟合
- 最终模型输入：Pre_(21维) + Target_(9维one-hot编码4种织构方案) + Process_(4维：温度/时间/H₂/Ar) = 34维特征向量

## 贝叶斯优化：代理模型与采集函数
### 贝叶斯优化循环：高斯过程代理模型 + EI采集函数驱动序贯最优搜索
- [image: bo_cycle.png]
- BO是一个序贯决策框架：在第t步，基于已有数据D_{1:t−1}构建代理模型，通过采集函数决定下一个最具信息量的实验点x_t
- 两个核心组件：（1）代理模型f(x)~GP(μ,k)，提供预测均值与不确定性；（2）采集函数α(x)，决定搜索方向
- 循环机制：初始实验数据→训练GP→最大化α(x)找到建议点→执行实验→更新数据→重复直到收敛
- 与传统优化对比：不需要目标函数的解析导数、不假设多项式形式、每一步都量化预测不确定性、实验次数显著减少

### GPR代理模型构建（上）：34维特征空间下的Matern核函数选择
- 完整特征空间：Pre_(21维EBSD统计) + Target_(9维织构方案one-hot) + Process_(4维工艺参数) → TARGET_Yield（织构产率）
- 核函数构成：ConstantKernel × Matern(ν=2.5, ARD=True) + WhiteKernel
- ConstantKernel：控制函数幅值的先验方差，自动从数据学习
- 选择Matern ν=2.5而非RBF的原因：Matern核的二阶可微性匹配物理过程的平滑性假设，同时避免RBF的"过度光滑"——Matern允许函数在局部有更真实的粗糙度，退火行为中可能存在局部突然变化
- Matern 2.5 相关性衰减形式：k(d) = σ²(1 + √5·d + 5/3·d²)exp(−√5·d)，其中 d=||x−x'||_θ
- WhiteKernel：显式建模观测噪声（实验重现性误差±2-3%），防止GP过度拟合单一数据点

### GPR代理模型构建（下）：ARD自动相关性确定与长度尺度约束
- ARD机制：每个特征k有独立的长度尺度l_k，核函数中d(x,x')² = Σ_k (x_k−x'_k)²/l_k²
- 长度尺度的物理含义：l_k小→相关性随该特征变化快速衰减→该特征对预测影响大（高重要性）；l_k大→相关性缓慢衰减→影响小
- 因此，训练完成后查看各特征的l_k值即可自动获得特征重要性排序——无需额外permutation test
- 长度尺度边界约束（0.3, 5.0）：数据经StandardScaler归一化至≈[−3,+3]后，下界0.3防止单点过拟合；上界5.0确保工艺参数可见
- 关键修正记录：原上界100.0导致exp(−(Δx/100)²)≈1对所有工艺参数——模型完全忽略工艺影响，将全部权重放在Pre_特征上。修正到5.0后工艺参数重新获得合理的特征重要性

### 小样本下的模型评估：LOOCV策略与预测性能验证
- N≈44的小数据集：传统80/20训练测试切分不可靠（单次切分方差过大，不同切分结果差异显著）
- GPR的固有陷阱：对训练点.predict()返回精确插值（预测值=真实值，σ=0）——这不能用于评估，只是数学恒等式
- LOOCV（留一交叉验证）：每次留出一个样本，用其余N−1个训练并预测留出点；重复N次，得到N个独立预测
- 评估指标：Parity plot的R²衡量预测vs真实产率相关性、RMSE衡量绝对误差、95%置信区间覆盖检验预测不确定性是否合理
- 实验结果显示R²>0.85，表明GPR在仅~40个训练样本下已具备可靠的外推预测能力

### EI采集函数：Expected Improvement在探索与利用之间的自动平衡
- EI数学定义：EI(x) = E[max(f(x) − y_best, 0)]——在候选点x处，目标函数值超过当前最优值的期望
- GPR下的解析形式（GP的关键优势——EI可精确计算无需蒙特卡洛）：EI(x) = (μ(x)−y_best)·Φ(Z) + σ(x)·φ(Z)，其中Z = (μ(x)−y_best)/σ(x)
- 第一项 (μ−y_best)·Φ(Z)：当预测均值μ高时EI大——驱动exploitation，在已知高产率区域附近精细搜索
- 第二项 σ·φ(Z)：当预测不确定性σ高时EI大——驱动exploration，向未知区域拓展
- 关键性质：EI参数自由（无手动exploration权重），从数据的μ和σ自动判断应该探索还是利用——这是GP+EI组合的核心优雅之处

### 两步优化策略：Latin Hypercube全局粗搜 + L-BFGS-B局部精炼
- y_best选取策略：（1）按织构方案分组，各方案使用局部y_best而非全局最大值——不同方案产率不可比较；（2）Winsorized P95替代raw max——防止单个异常高产率点主导整个搜索方向
- 第一步：100,000个Latin Hypercube采样点均匀覆盖4维工艺空间→计算所有候选点的EI→取Top-5作为精炼起点
- LHS优势：相比纯随机采样，LHS保证每个维度均匀分层，在4维空间中以更少样本达到全局覆盖
- 第二步：对Top-5分别启动L-BFGS-B梯度优化器→在局部精确爬升至EI极大值点→输出最高EI点作为建议下一实验
- 物理约束Ar≥2H₂在LHS采样阶段通过拒绝采样直接过滤，确保所有建议点物理可行

### 多任务织构方案编码：4种目标织构的联合优化框架
- 4种织构目标方案各激活多个晶体取向（multi-hot encoding），同一Pre_+Process_特征在不同方案下有不同产率
- 方案1 <103>型(橙色系)：(1,0,3)+(1,0,2)+(3,0,1)、方案2 <114>型(粉紫色系)：(1,1,4)+(1,1,5)+(1,0,5)
- 方案3 <124>型(混合色)：(1,2,4)+(1,2,5)+(2,1,4)、方案4 自定义组合：(1,0,3)+(1,1,4)+(1,2,4)
- 每实验数据扩展为4行（对应4个方案），每行共享相同的Pre_和Process_，但Target_编码和Yield不同
- 完整优化流程：EBSD原始数据→21维Pre_提取→合并训练数据→GPR训练(LOOCV验证)→EI最大化(两步优化)→建议下一实验→执行→更新模型→收敛判据(不确定性衰减)

## 总结
### 核心结论与后续工作
- 数据降维策略有效：10⁴像素→21维特征，保留织构取向偏好和位错密度统计轮廓的关键信息，Pearson初筛进一步缓解小样本过拟合
- GPR+ARD核适合小样本退火建模：Matern 2.5提供物理合理性，ARD自动识别关键预测因子，长度尺度边界0.3–5.0防止工艺参数被忽略
- EI采集函数驱动优化收敛：LOOCV R²>0.85，不确定性随样本增加单调递减，两步优化（LHS+L-BFGS-B）平衡全局探索与局部精炼
- 后续工作：（1）扩充文献挖掘数据以增加训练样本量和材料多样性；（2）探索多目标Trade-off优化（同时优化多个织构方案）；（3）引入实验成本模型，使BO在建议下一实验时考虑时间/材料约束
