# 技能名称
pure-ni-reader

# 技能描述
从原始文献中提取纯镍退火的工艺参数与初始/最终状态的 EBSD 表征结果，并严格按照本地指定的目录结构拆分为三个独立的数据文件。

# 执行逻辑
接收到原始文本后，严格按照以下步骤执行：

1. 参考数据结构
在处理数据前，你必须查阅并严格参照本地路径 `D:\毕业设计\织构数据\数据总结\1` 下的数据表结构。你的输出必须与该目录下的文件格式保持绝对一致。

2. 数据提取与文件拆分
你必须将提取到的信息整理并分别输出为以下三个文件的数据结构：

- condition.csv（工艺条件文件）
  提取并换算退火工艺参数，表头必须严格为：`time(h),temprature(℃),H2,Ar`。
  注意：退火温度（特别是1200℃-1400℃区间）必须换算为摄氏度。如气体条件（H2, Ar）描述不足或未提及，对应字段必须留空，绝不允许捏造数据。

- pre.csv（退火前初始状态文件）
  提取文献中关于纯镍样品的初始状态数据（如单晶镍箔的初始织构、退火前的 GND 密度或 IPF 特征）。结构需尽量对齐本地 `pre.csv` 中的 `Index, X, Y, IPF Coloring, GND Density` 格式，若文献未提供详细点位数据，则提取整体特征描述并留空不支持的列。

- done.csv（退火后最终状态文件）
  提取经过上述条件退火后的最终 EBSD 表征结果，重点提取最终的织构演变结果与 GND 密度。数据结构需严格对齐本地 `done.csv` 格式。

3. 输出要求
完成数据提取后，直接展示这三个文件应有的结构化内容（或直接生成相应的 CSV 文本），确保能够直接被读取或合并进入现有的贝叶斯优化数据集。
