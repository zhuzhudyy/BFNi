# 技能名称
pure-ni-annealing-scraper

# 技能描述
专门用于检索和抓取关于纯镍（Pure Nickel）退火工艺学术文献的技能。该技能确保获取的数据必须包含实验工艺参数以及EBSD（电子背向散射衍射）表征结果。

# 执行逻辑与约束
当触发文献抓取任务时，必须遵循以下筛选准则：

1. 材质限制
抓取的目标对象必须限定为纯镍（Pure Nickel），不得包含镍基合金（如Ni-Cr, Ni-W等）或复合材料，除非该文献是作为纯镍性能对比的对照组。

2. 核心数据要求
每篇抓取的文章必须包含以下两类关键信息：
- 工艺条件：必须明确记录退火温度（800℃——1400℃）、退火时间、升温速率、气体条件。
- EBSD数据：必须包含EBSD表征结果，包括晶粒取向图（IPF），几何必须位错密度（GND）图，最好有二者原始数据。

3. 检索策略
优先使用组合关键词进行检索：
- 关键词组A：Pure Nickel, Annealing, Recrystallization, Grain Growth.
- 关键词组B：EBSD, Orientation Imaging Microscopy, Texture Evolution, Grain Boundary.
- 必选逻辑：(关键词组A) AND (关键词组B).

4. 验证与清洗步骤
- 预检：在下载或完整提取前，先阅读摘要（Abstract）和实验部分（Experimental Section），确认是否存在EBSD分析。
- 排除：排除仅有拉伸测试或SEM形貌而无晶体学取向分析的文献。
- 排除：排除仅讨论液态金属而不涉及固态退火组织的文献。

5. 数据交付
- 提取文章的标题、作者、年份及DOI。
- 将抓取到的原始文本或PDF提取内容保存为本地文件。
- 标记该文献对应的具体退火工艺参数，以便后续读取智能体（Reader Agent）进行结构化处理。

# 交付标准
任务完成时应报告：已找到符合纯镍退火、包含工艺条件且具备EBSD数据的文献数量，并提供保存路径。
